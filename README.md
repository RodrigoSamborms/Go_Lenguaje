# Go_Lenguaje
Para instalar el compilador revise el documento de [Guia de Instalacion](./instalacion.md). Nota se trabajara en WSL Debian, asi que asegure carga una terminal en VS code con WSL Debian al manejar este repositorio, en la terminal de PowerShell del VS code escriba:
```
PS C:\Users\sam> WSL -d Debian
```
Obervara como cambia el prompt a uno tipico de una terminal de Linux:
```
UserName@HOSTMachine:/mnt/c/Users/sam$ 
```
Asi podra ejcutar los archivos escrito en este lenguaje:
```
$ go run <Directorio/NombreArchivo>
```
